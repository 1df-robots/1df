Welcome to 1 Degree Freedom Robotics

Modular Smart Actuation Systems

1 Degree Freedom robotics is founded in June 2019 with a vision to help Robotic solution providers solving the problems in motion control.
We are a team of passionate engineers from different disciplines working on actuation solutions. We develop modular systems that can be easily integrated into various types of robots. Robotic solution providers spend a significant amount of time and resources to select and integrate various combinations of actuation components available in the market. We are solving that problem by integrating motor, gearbox, driver, controller and communication system into a plug and play module that can be easily controlled by our software APIs.

Modes in which the Multi-axis controller can be operated

Manual Mode

The Host computer would stream the required velocity commands to the multi-axis controller which would relay the command to the actuator. Maintaining the trajectory would be the responsibility of the host computer.

Trajectory mode

The kinematic configuration of the actuators has to be setup through a Software module. The host computer would send a command to the multi-axis controller specifying the start and end of a navigation position. The multi-axis controller would generate the trajectory profile that would ensure the position, torque and velocity coordination of the actuators.
