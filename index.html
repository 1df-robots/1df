Smart Robotic Joints and Multi-Axis Motion Controllers

Compact Robotic joint drive system with integrated motor control electronics in a compact form factor. The Actuator is a combination of motor, closed loop control system, gear assembly for generating higher torque and a feedback system. Watch video

Modes in which the Multi-axis controller can be operated

Manual Mode

The Host computer would stream the required velocity commands to the multi-axis controller which would relay the command to the actuator. Maintaining the trajectory would be the responsibility of the host computer.

Trajectory mode

The kinematic configuration of the actuators has to be setup through a Software module. The host computer would send a command to the multi-axis controller specifying the start and end of a navigation position. The multi-axis controller would generate the trajectory profile that would ensure the position, torque and velocity coordination of the actuators.
